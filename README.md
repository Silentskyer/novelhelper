## 目前版本說明

目前版本為純前端靜態網站：

- 使用者可在頁面輸入 Gemini API Key
- 可生成小說靈感
- 可生成角色設計建議
- 支援複製結果
- 支援回應風格與篇幅設定

## 使用方式

直接開啟 `index.html` 即可使用。

若 Gemini 請求失敗，請確認：

- API Key 是否有效
- Gemini API 是否已在 Google AI Studio 啟用
- 瀏覽器是否允許對外連線

## 後續建議

下一步建議升級為正式前後端架構，避免在前端直接暴露 API Key，例如：

- 部署：Vercel / Netlify + Serverless Functions
